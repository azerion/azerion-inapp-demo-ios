# BlueStack iOS SDK Demo Apps

Demo applications showcasing the integration and features of the BlueStack iOS SDK.

## Available Demos

### [BlueStack iOS SDK Demo (UIKit)](Swift/BlueStackDemo)
A comprehensive iOS demo app built with Swift and UIKit, demonstrating:
- Banner ads (Standard, Large, Leader, Full, Sticky)
- MREC (Medium Rectangle) ads
- Interstitial ads
- Rewarded video ads
- Coordinator pattern architecture
- Consent Management Platform (CMP) integration
- Custom UI components and logging

**Tech Stack:** Swift, UIKit, Storyboards, CocoaPods  
**Minimum iOS:** 13.0

[View Documentation →](Swift/BlueStackDemo/README.md)

---

## Getting Started

Navigate to the demo directory and follow the setup instructions in its README:

```bash
cd Swift/BlueStackDemo
```

## Requirements

- Xcode 16.0 or later
- CocoaPods
- iOS 13.0+ deployment target
